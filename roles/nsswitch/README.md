Nss
===

Manage /etc/nsswitch.conf entries.

## Variables

| Variable           | Default                | Description                          |
|--------------------|------------------------|--------------------------------------|
| `nss_enabled`      | `true`                 | Active ou désactive la gestion NSS   |
| `nsswitch_file`    | `/etc/nsswitch.conf`   | Chemin du fichier nsswitch.conf      |
| `nsswitch_config`  | voir defaults          | Dictionnaire des entrées nsswitch    |
